# 📘 Playwright Test DWS

## 🚀 Overview
This project is an end-to-end automation testing framework built using **Playwright**. It is designed to test web applications efficiently with modern testing practices, ensuring reliability, speed, and cross-browser compatibility.

---

## 🧰 Tech Stack
- Playwright
- JavaScript / TypeScript
- Node.js
- HTML Reports / Playwright Reporter

---

## 📂 Project Structure
```
PlaywrightTestDWS/
│── tests/                # Test scripts
│── pages/                # Page Object Model (POM)
│── utils/                # Utility/helper functions
│── playwright.config.js  # Configuration file
│── package.json          # Dependencies & scripts
```

---

## ⚙️ Installation

### 1. Clone the Repository
```bash
git clone https://github.com/teche74/PlaywrightTestDWS.git
cd PlaywrightTestDWS
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Install Playwright Browsers
```bash
npx playwright install
```

---

## ▶️ Running Tests

### Run All Tests
```bash
npx playwright test
```

### Run Specific Test
```bash
npx playwright test tests/example.spec.js
```

### Run in Headed Mode
```bash
npx playwright test --headed
```

---

## 📊 Test Reports

After execution, view reports using:
```bash
npx playwright show-report
```
